# ☁ Cloudflare IP Scanner & V2Ray Tester
### ساخته شده توسط عرشیا کوهیان محمد آبادی

---

## ✅ امکانات

### مود ۱ — تست Ping (بدون کانفیگ)
- وارد کردن رنج IP به فرمت CIDR (مثلاً `104.16.0.0/16`)
- استخراج خودکار همه IP ها
- تست TCP Ping همزمان (multi-thread)
- نمایش ping هر IP به میلی‌ثانیه
- مرتب‌سازی بر اساس سرعت
- ذخیره نتایج به فایل CSV/TXT

### مود ۲ — تست V2Ray / Xray
- بارگذاری کانفیگ JSON یا لینک share
- جایگزینی خودکار IP در کانفیگ
- اجرای Xray با routing صحیح
- تست اتصال واقعی از طریق پروکسی (بدون curl)
- لاگ رنگی نتایج

## 📋 راهنمای استفاده

### مود Ping
1. رنج CIDR را وارد کنید (مثال: `104.16.0.0/13`)
2. حداکثر تعداد IP و تعداد Thread را تنظیم کنید
3. دکمه «شروع اسکن» را بزنید
4. نتایج به ترتیب ping نمایش داده می‌شوند

### مود V2Ray
1. در تب «کانفیگ» لینک vless/vmess/trojan خود را paste کنید و Parse کنید
2. به تب «تست Ping» بروید و اسکن را انجام دهید
3. در تب «تست V2Ray/Xray» روی «از Ping» کلیک کنید
4. مسیر xray.exe را تنظیم کنید
5. «شروع تست» را بزنید
